# 米游社 WIKI 壁纸批量提取与下载助手

一款专为米游社 WIKI（《原神》观测枢、《崩坏：星穹铁道》开拓者笔记、《崩坏3》大百科等）设计的油猴（Tampermonkey）脚本。旨在帮助玩家快速、无损地批量提取并下载页面中的精美壁纸。

---

## 🚀 核心特性

* 🎨 **原画无损画质**：自动剥离阿里云 OSS 缩略图参数，100% 还原 4K/高清无水印原画。
* 🏷️ **原始标题命名**：智能 DOM 层级分析，自动将壁纸在页面下方的文字作为文件名（例如：`「与我不曾知晓的你」系列壁纸.png`），告别混乱的代码命名。
* 🔍 **分辨率宽度过滤**：内置最小宽度筛选输入框（默认为 1000px），智能过滤头像、小图标和表情包。
* ⚡ **无弹窗一键直下**：自动模拟滑动页面进行懒加载，摒弃繁琐的弹窗确认，点击按钮直接进入后台单线程队列静默下载。
* 🛡️ **同名冲突防护**：自动检测重名壁纸并追加递增序号（如 `_2`, `_3`），防止重复文件名被覆盖。
* ⚙️ **自动提取模式**：支持“页面载入后自动执行”，免去每次手动点击的繁琐。

---

## 📦 安装指南

在安装此脚本前，请确保您的浏览器已安装了 [Tampermonkey (油猴) 扩展插件](https://www.tampermonkey.net/)。

### 👉 [点击此处一键安装/更新脚本](https://github.com/removeshort-prog/miyoushe-wiki-wallpaper-helper/raw/refs/heads/main/miyoushe-wallpaper-helper.user.js)

## 🛠️ 使用说明

1. 使用电脑浏览器打开米游社 WIKI 的任意壁纸或插画合集页面。
2. 页面加载完成后，右下角会自动悬浮一个蓝色小图标：`📥`。
3. 点击图标展开控制面板：
   * 选择您喜欢的**命名规则**（推荐使用“优先使用原始标题”）。
   * 设定**过滤最小宽度**（如输入 `1200`，则小于 1200 像素宽度的图将不被提取）。
4. 点击 **🚀 开始提取（自动滚动）**，脚本会模拟自动下滑，请静待其回滚至顶部。
5. 提取完成后，点击 **💾 一键批量下载** 即可将高清大图保存至本地。

---

## ⚠️ 注意事项

* **多文件下载权限**：第一次点击批量下载时，浏览器地址栏右侧或上方可能会提示 **“允许此网站下载多个文件”** 的弹窗申请。请务必点击 **【允许】**，否则后续图片下载请求会被浏览器拦截。
* **默认存储位置**：脚本下载的文件默认会保存在您浏览器的“下载”目录中，并自动归类至 `Miyoushe_Wiki_Wallpapers` 子文件夹内。
